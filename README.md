# common-recycler-view
common-recycler-view는 Multi view type를 지원하는 RecyclerView의 예제 프로젝트입니다. 서로 다른 리싸이클러뷰의 아이템들을 뷰타입으로 분류하고 정의하여, 단 하나의 Recycler View Adapter만으로 다양한 아이템을 나타냅니다.
